# 💅 Dani Rizos Nail's — Sitio Web Estático

Este es el sitio web oficial de **Dani Rizos Nail's**, un salón de belleza especializado en arreglos de uñas con un enfoque en el detalle, el orden y el estilo. El sitio permite a los clientes explorar el portafolio de trabajos, conocer los servicios detallados y agendar citas directamente a través de un sistema de mensajes preconfigurados para WhatsApp.

## 🚀 Características Principales

- **Diseño Editorial y Premium:** Una estética minimalista y moderna que refleja la calidad del servicio.
- **Portafolio Interactivo:** Carrusel de imágenes con los trabajos más destacados y categorías.
- **Catálogo de Servicios:** Página detallada con precios, tiempos estimados y características de cada servicio (Gel semipermanente, Acrílico escultural, Nail art).
- **Sistema de Reserva por WhatsApp:** Formulario inteligente que genera un mensaje estructurado con emojis para facilitar la comunicación entre la clienta y el salón.
- **Navegación Inteligente (Scroll Spy):** El menú de navegación resalta automáticamente la sección que el usuario está viendo.
- **Totalmente Responsivo:** Optimizado para dispositivos móviles, tablets y computadoras de escritorio.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica para SEO y accesibilidad.
- **Vanilla CSS3:** Estilizado personalizado con variables, Flexbox, Grid y animaciones fluidas.
- **JavaScript (Vanilla):** Lógica del carrusel, menú móvil, validación del formulario de reserva y sistema de scroll spy.

## 📂 Estructura del Proyecto

```text
sitio-estatico/
├── img/                # Imágenes de portafolio, servicios y logo
├── index.html          # Página principal (Hero, Trabajos, Reserva)
├── servicios.html      # Catálogo detallado de servicios
├── styles.css          # Estilos globales y componentes
└── script.js           # Lógica interactiva del sitio
```

## 💻 Cómo Ejecutar Localmente

Como es un sitio estático, puedes probarlo de dos maneras:

1. **Abrir directamente:** Simplemente haz doble clic en el archivo `index.html` en tu navegador.
2. **Servidor local (Recomendado):** Para asegurar que todas las funciones carguen correctamente, puedes usar un servidor local. Si tienes Python instalado, ejecuta:
   ```bash
   python3 -m http.server 8080
   ```
   Luego abre tu navegador en `http://localhost:8080`.
